# Football Player Management System

A Python-based console application for managing football players and teams using text files.

## Features
- Add a player
- Update player statistics
- Show statistics
- Add a team
- Display all players
- Show players without teams
- Delete a player
- Delete a team and its players
- Save statistics to a file

## Technologies
- Python
- File handling
- Functions
- Menu-driven interface

## Files
- `main.py` → main program
- `players.txt` → player records
- `teams.txt` → team records
- `statistics.txt` → saved statistics output

## How to Run
1. Make sure Python is installed
2. Run:
   ```bash
   python main.py
